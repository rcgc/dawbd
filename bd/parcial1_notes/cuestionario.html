¿Qué es entonces una base de datos - DB?

Un conjunto de registros almacenados y relacionados lógicamente.


¿En qué casos es conveniente usar bases de datos?

Cuando se requiere procesar la información en acciones tales como: filtrar información, consultar (rápidamente) datos específicos, establecer restricciones en las entradas de información, etc.


¿Qué es un sistema de gestión de base de datos - DBMS?

Un software capaz de gestionar los archivos de múltiples bases de datos, que en conjunto con el sistema operativo, ambos facilitarán la tarea de gestionar lo anteriormente mencionado. Finalmente, el sistema gestor de base de datos se encargan de la privacidad, la integridad, la seguridad de los datos y la interacción con el sistema operativo. Proporciona una interfaz entre los datos, los programas que los manejan y los usuarios finales.


Enlista y explica las funciones/responsabilidades que tienen los DBMS.

  1. Implantación de la integridad: los valores de los datos que se almacenan en la base de datos deben satisfacer ciertos tipos de restricciones de consistencia.
  2. Implantación de la seguridad: no todos los usuarios de la base de datos necesitas tener acceso a todo su contenido. Es trabajo del gestor de la base de datos hacer que se cumplan estos requisitos de seguridad.
  3. Copia de seguridad y recuperación: detectar tales fallos y restaurar la base de datos al estado que existía antes de ocurrir el fallo. Esto se lleva a cabo normalmente a través de la iniciación de varios procedimientos de copias de seguridad y recuperación.
  4. Control de concurrencia: controlar la interacción entre los usuarios concurrentes.

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En la preparatoria cuando realicé una página web en la que mostraba los productos de una tienda de materias primas, en pocas palabras, realicé un catálogo con la restricción de que sólo mostraba productos pero no permitía realizar compras online. Finalmente, concluyo en que el manejo de una base de datos podría haber sido de ayuda sólo para actualizar el catálogo y así ahorrar el hecho de programar cada que se introdujera un nuevo producto.
