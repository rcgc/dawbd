Js, JQuery & AJAX
Herramienta Descripción
JavaScript Es un lenguaje de programación conocido por ser interpretado por los navegadores de páginas web
JQuery Es una librería de JavaScript, escrita por John Resig que surgió en el 2006 para solucionar las diferencias al manipular HTML y escuchar eventos en los navegadores. Aunque ya existían intentos similares en JavaScript, una de las genialidades de jQuery era que utilizaba los mismo selectores CSS para ubicar y manipular los diferentes elementos del DOM.
AJAX AJAX es una técnica que permite, mediante programas escritos en Javascript, que un servidor y un navegador intercambien información, posiblemente en XML, de forma asíncrona

XMLHttpRequest.readyState

La propiedad XMLHttpRequest.readyState retorna el estado en que un cliente XMLHttpRequest está. Un cliente XHR existe en uno de los siguientes estados:


XMLHttpRequest.readyState
Value State Description
0 UNSET The XMLHttpRequest client has been created, but the open() method hasn't been called yet.
1 OPENED open() method has been invoked. During this state, the request headers can be set using the setRequestHeader() method and the send() method can be called which will initiate the fetch.
2 HEADERS_RECEIVED send() has been called and the response headers have been received.
3 LOADING Response's body is being received. If responseType is "text" or empty string, responseText will have the partial text response as it loads.
4 DONE The fetch operation is complete. This could mean that either the data transfer has been completed successfully or failed.

Alternativas gratuitas
Herramienta Descripción Ventajas Desventajas
UmbrellaJS Es una librería fuertemente influenciada por jQuery, pero mucho más comprimida y limitada Sólo pesa 4kb comprimida, por lo tanto su tiempo de carga es apto para dispositivos móviles Limitada a la manipulación DOM, eventos y Ajax
Webix Es un framework que nos provee una forma fácil de crear elementos comunes de una interfaz de usuario, comparte elementos de uso comun con JQuery La difusión de información acerca de este framework permite que siempre existan tutoriales con datos recientes Hay elementos tales como date-range pickers, selectors, shedulers que para ser usados requieren de la versión premium
Zepto.js Librería tipo minimalista Compatible con JQuery y especialmente dirigida a entornos muy exigentes en rendimiento .
XUI Librería tipo minimalista enfocada para dispositivos móviles Su sintaxis es heredera de JQuery Al ser relativamente nueva (lanzada desde 2008) cuenta con poca madurez y tutoriales
Sizzle Es una librería selectora, es decir, sólo necesitamos la capacidad de utilizar los selectores de tipo CSS y asociar después un determinado evento sobre los mismos a mano Completamente independiente. No requiere de ninguna dependencia, altamente extensible mediante una sencilla y potente API y sólo pesa 4 kb .
Sprint.js Es un framework compatible con los conocimientos de JQuery Más rápido que JQuery y Zepto.js No tiene soporte para versiones viejas de navegadores
